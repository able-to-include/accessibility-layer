# accessibility-layer
A context-aware accessibility layer that, by being integrated with existing and future ICT tools, can improve the day-to-day life of people with IDD by understanding their surroundings and helping them to interact with the information society. Test.

This API is public and can be used as it is explained at the <a href="http://al.abletoinclude.eu/doc/">ABLE Accessibility Layer Documentation</a>.
